Улучшен вывод ошибок сборки для `start` и `build`. Ошибки группируются по категориям (TypeScript, 
Module, CSS, Module Federation, Configuration и тд.), для известных типов ошибок выводятся подсказки по 
исправлению (проверка пути или установка пакета, `tsc --noEmit` для ошибок типизации, увеличение лимита 
памяти Node.js и тд)
